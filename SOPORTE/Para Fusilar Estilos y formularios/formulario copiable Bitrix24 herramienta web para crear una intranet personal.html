Bitrix24 herramienta web para crear una intranet personal

 Por Gesivet Publicado en: Herramientas web El 22 de octubre de 2012

Bitrix24, es una aplicación web que te permite crear y administrar tareas en equipo, compartir archivos y obtener información sin tener que configurar cualquier solución de trabajo de gestión especializada, e incluso si tienes un iOS (iPhone y iPad) de gestión.

Al registrar una cuenta con el servicio, puedes crear un espacio de trabajo personal donde cada uno que elijas puedes colaborar en las tareas. Cada área de trabajo cuenta con su propia URL dentro de la aplicación web, el servicio pretende construir una intranet, donde las personas pueden comunicarse unos con otros, se ayudan mutuamente con los proyectos, supervisan su mutuo progreso, comparten archivos e inician de 1-a-1 discusiones.

Para utilizar el servicio, lo primero que se requiere es crear una cuenta. Introduce el nombre que deseas para la intranet, el nombre de la empresa y sus datos personales para la creación de la cuenta. Una vez creada la cuenta, puedes iniciar sesión en él utilizando su dirección de correo electrónico y contraseña.

El servicio ofrece dos cuentas gratuitas y de pago para los usuarios. La cuenta gratuita tiene un límite de 5 GB de datos y 12 participantes en la intranet. La comparación completa de las características, incluidos en los paquetes gratuitos y de pago, se puede encontrar en la página del producto.
